# 记账应用分类调研报告

## 调研对象

1. **Cookie记账** - 中国区流行，用户评价高
2. **鲨鱼记账** - 极简风格，用户基数大
3. **随手记** - 老牌记账应用，功能全面
4. **MoneyWiz** - 国际知名，支持多平台
5. **YNAB** - 国外流行的预算记账应用

## 实施建议

### 第一阶段：核心分类（MVP）
保持现有16个支出分类 + 8个收入分类，确保基础功能完整。

### 第二阶段：扩展分类（V1.1）
新增高频使用的分类：
- 支出：水电燃气、通讯、服饰鞋包、孩子育儿、车辆相关
- 收入：加班补贴、利息分红、租金收入、创作稿费

### 第三阶段：完整分类（V1.2）
补充所有建议分类，支持用户自定义分类和子分类。

### 用户自定义
- 支持用户新增自定义分类
- 支持修改分类名称和图标
- 支持隐藏不使用的分类
- 支持分类排序
