## 📋 使用建议

### 1. 下载方式
- 访问对应的 Thiings URL
- 点击图标查看大图
- 右键保存或使用下载按钮

### 2. 格式选择
- **推荐格式：** PNG（透明背景）
- **备选格式：** SVG（矢量，适合缩放）
- **尺寸建议：** 512x512px 或 1024x1024px

### 3. 风格统一
所有图标来自同一平台，确保：
- 3D渲染风格一致
- 光影效果统一
- 色彩饱和度相近

### 4. 在Numi中的应用
```swift
// 示例：在SwiftUI中使用
Image("thiings_acai_bowl")
    .resizable()
    .aspectRatio(contentMode: .fit)
    .frame(width: 40, height: 40)
```
