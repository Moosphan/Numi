## 使用建议

### 1. 尺寸调整
原始图标尺寸较大（约1-2MB），建议在App中使用时：
- 列表图标：64x64 或 80x80 像素
- 详情图标：128x128 或 256x256 像素
- 高清显示：提供 @2x 和 @3x 版本

### 2. 样式处理
- 可添加圆角矩形遮罩
- 根据Numi主题色调整色调
- 保持阴影和光影效果

### 3. 在SwiftUI中使用
```swift
Image("acai-bowl")
    .resizable()
    .aspectRatio(contentMode: .fit)
    .frame(width: 40, height: 40)
```

### 4. 在Asset Catalog中使用
1. 将图标拖入 Asset Catalog
2. 设置合适的尺寸变体
3. 在代码中引用名称
